Quem sou eu?

Olá, sou o Kayo. Atuo como Desenvolvedor Fullstack e estou cursando o sexto período de Sistemas de Informação na UNINASSAU - Unidade Graças, unindo a prática do mercado com uma base acadêmica sólida. Meu foco principal é o Frontend, no qual busco criar interfaces intuitivas e que proporcionem uma ótima experiência ao usuário. Tenho também total confiança no Backend para construir e garantir que toda a estrutura funcione perfeitamente nos bastidores. Busco ser uma pessoa bastante comunicativa, podendo me expressar bem em inglês(C1) e estou iniciando meus estudos em Libras, por acreditar em uma web mais acessível.

Em projetos, tomo a iniciativa de assumir a frente sempre que necessário. Meu estilo de liderança é focado em manter a calma sob pressão e em realizar uma análise estratégica da equipe. Identifico os pontos fortes de cada membro para otimizar o desempenho e direciono os esforços para as melhorias necessárias, sempre buscando incluir uma margem de tempo no cronograma para possíveis imprevistos em todas as fases do projeto. Minha aptidão natural para a liderança, que foi reconhecida e posta em prática quando atuei como líder de sala na faculdade.

Por fim, também tenho interesse em aréas como Design no qual aprimora a qualidade visual das minhas entregas, Marketing e Dados me ajuda a entender o que os usuários realmente precisam. Além de me ver lecionando, pois compartilhar conhecimento é a melhor forma de consolidar o aprendizado.

Cloud Computing

A minha primeira experiência com o Supabase me mostrou o poder e a necessidade da nuvem. Para aprofundar nesse universo, estou estudando AWS com Python, focando em como construir, hospedar e escalar aplicações de forma eficiente e segura na principal plataforma de cloud do mercado.

Estou cada vez mais fascinado pelo mundo da segurança digital. Meus estudos incluem os tipos de ataques e atacantes, como os vírus operam e, o mais importante, as melhores práticas e arquiteturas (síncronas e assíncronas) para criar sistemas mais seguros e protegidos.

Machine Learning

A curiosidade sobre como as IAs funcionam "por baixo dos panos" me levou a estudar os fundamentos de Machine Learning. No momento, estou focado na teoria por trás das redes neurais para compreender a base do que move a inteligência artificial hoje.

Boas Práticas e Arquitetura de Software

Acredito que a qualidade do código é o que define um bom profissional. Por isso, estou lendo e aplicando os conceitos do livro "Código Limpo" (Clean Code) de Robert C. Martin em todos os meus projetos, buscando sempre escrever um código que seja não apenas funcional, mas também legível, manutenível e bem arquitetado.

SK

Proposta: Meu primeiro projeto Fullstack completo, desenvolvido de forma solo. O SDK é um e-commerce farmacêutico criado como uma evolução direta do meu primeiro projeto, o "Lumatitaka". Desta vez, o desafio foi construir toda a arquitetura, do frontend ao banco de dados, utilizando um stack moderno e containerizado.

Principais Desafios e Aprendizados:

  • Criação de uma aplicação completa, integrando um frontend reativo em React com uma API RESTful robusta em Express/Node.js e um banco de dados relacional MySQL.
  • Utilização do Docker para containerizar toda a aplicação, garantindo um ambiente de desenvolvimento consistente e portátil, rodando em uma VM Linux. Foi um aprendizado prático imenso sobre o poder da containerização.
  • Responsável por todas as etapas, desde a modelagem do banco de dados e desenvolvimento do backend até a criação da interface no frontend.
  • O projeto está atualmente sendo atualizado para utilizar armazenamento em nuvem, demonstrando a aplicação de práticas modernas de cloud.

Tecnologias: Docker, React, Tailwind CSS, Express, Node.js, MySQL, Linux VM

Repositório: Veja no meu github

Loja

Proposta: Desenvolvido no programa Bolsa Futuro Digital (Softex), este projeto é um e-commerce com temática gamer, onde usuários podem comprar roupas, montar equipes de e-sport e fazer upgrades em seus computadores. O diferencial deste projeto foi a minha abordagem: optei por usar tecnologias fundamentais para poder atuar como mentor de um colega de equipe que estava tendo seu primeiro contato com o desenvolvimento web.

Desafios e Aprendizados:

  • Meu principal papel foi guiar e ensinar um novo desenvolvedor, traduzindo conceitos de programação em tarefas práticas e tirando todas as suas dúvidas, desde o HTML básico até a lógica com JavaScript.
  • A decisão estratégica de usar HTML, CSS e JavaScript puros para solidificar não só o aprendizado do meu colega, mas também minha própria capacidade de explicar e dominar os fundamentos.
  • A experiência de ensinar foi um aprendizado imenso em comunicação, paciência e empatia, reforçando o valor de construir conhecimento em equipe.

Tecnologias: HTML, CSS e JavaScript
